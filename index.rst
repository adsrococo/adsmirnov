=========================
Визуализация результатов
=========================

Serpent может готовить графические output-файлы для визуализации геометрии расчетной модели и результатов моделирования. Графические файлы готовятся в широко поддерживаемом формате PNG. Для создания графического вывода Serpent должен быть скомпилирован с графическими опциями и связан с графической библиотекой GD (см. Инструкции по компиляции исходного кода).
